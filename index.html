Materi Hardware

Hardware adalah perangkat keras komputer yang dapat dilihat dan disentuh secara langsung. Hardware merupakan bagian fisik dari sistem komputer yang memiliki peran penting dalam menjalankan berbagai proses kerja komputer. Tanpa adanya hardware, komputer tidak dapat digunakan sebagaimana mestinya.


Secara umum, hardware berfungsi untuk mendukung proses input, proses, output, dan penyimpanan data. Oleh karena itu, setiap perangkat keras memiliki fungsi yang berbeda beda tetapi saling mendukung satu sama lain dalam sistem komputer.

Contoh Perangkat Hardware

KEYBOARD: Digunakan untuk memasukkan huruf, angka, simbol, maupun perintah tertentu ke dalam komputer.

MOUSE: Berfungsi untuk menggerakkan pointer pada layar, memilih menu, dan menjalankan perintah klik.

MONITOR: Perangkat output yang berfungsi untuk menampilkan hasil proses komputer dalam bentuk visual.

CPU: “Otak komputer” yang berfungsi sebagai pusat pengolahan data dan pengendali utama seluruh proses kerja.

PRINTER: Perangkat output yang digunakan untuk mencetak dokumen atau data dari komputer ke media kertas.

Materi Software

Software adalah perangkat lunak komputer yang tidak dapat dilihat secara fisik, tetapi dapat digunakan dan dijalankan oleh pengguna. Software merupakan kumpulan instruksi atau program yang berfungsi untuk mengatur dan mengendalikan kerja hardware agar dapat berjalan dengan baik.


Tanpa adanya software, hardware tidak dapat digunakan secara maksimal karena tidak ada perintah yang mengatur proses kerja komputer. Software memiliki peran penting dalam menjalankan berbagai aktivitas pada komputer.


Secara umum, software berfungsi untuk mengatur proses kerja komputer, menjalankan perintah dari pengguna, serta mengelola data menjadi informasi yang berguna.

Contoh Perangkat Software

SISTEM OPERASI: Software utama yang mengatur aktivitas komputer (Windows, Linux, macOS).

SOFTWARE APLIKASI: Membantu tugas tertentu seperti mengetik atau browsing (Ms. Word, Google Chrome).

SOFTWARE UTILITAS: Merawat kinerja komputer (Antivirus, CCleaner, WinRAR).

PEMROGRAMAN & DESAIN: Software khusus untuk berkarya (VS Code, Photoshop, AutoCAD).
